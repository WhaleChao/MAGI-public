# 刑事訴訟法證據能力分類規則（辯護方觀點）

本文件詳述被告方對卷證索引各類證據表示證據能力意見的完整邏輯。

## 核心法條

### 刑事訴訟法第159條第1項（傳聞法則）
被告以外之人於審判外之言詞或書面陳述，除法律有規定者外，不得作為證據。

### 刑事訴訟法第159條之1第2項（檢察官前之陳述）
被告以外之人於偵查中向檢察官所為之陳述，除顯有不可信之情況者外，得為證據。

辯護方立場：此類陳述雖依法「得為證據」，但被告有權表示不同意，要求法院依嚴格證明法則調查。

### 刑事訴訟法第155條第2項（合法調查原則）
無證據能力、未經合法調查之證據，不得作為判斷之依據。

辯護方立場：檢察官援引某證人之偵訊筆錄，但該證人尚未於審判中作證接受詰問，屬「未經合法調查」，不得作為判斷依據。

### 最高法院114年度台上字第3941號判決
重申被告之反對詰問權為訴訟上基本權利，偵訊筆錄若未經合法調查（傳喚證人到庭接受詰問），不得逕為判斷之依據。

## 分類規則（依優先序）

### Rule 1: 被告本人的任何筆錄
- 條件：section_type == 'defendant_self'
- 意見：`不爭執證據能力`
- 理由：被告本人的陳述不適用傳聞法則

### Rule 2: 審判筆錄 / 言詞辯論筆錄
- 條件：record_type in ('審判筆錄', '言詞辯論筆錄')
- 意見：`不爭執證據能力`
- 理由：審判中的陳述非審判外陳述，不屬傳聞

### Rule 3: 法院程序筆錄
- 條件：record_type in ('準備程序筆錄', '協商程序筆錄')
- 意見：`不爭執證據能力`
- 理由：法院程序的紀錄，非傳聞證據

### Rule 4: 訊問筆錄（區分法院階段 vs 偵查階段）
- **法院階段**（卷宗字號含「國審強處」或「訴字」）：
  - 意見：`不爭執證據能力`
  - 理由：法院階段的訊問屬審判中程序
- **偵查階段**（卷宗字號含「偵字」、「相字」、「他字」）：
  - 繼續適用下方 Rule 6-8 的傳聞法則判斷
  - 理由：偵查階段的訊問筆錄實質上等同偵訊筆錄

### Rule 5: 書證 / 物證
- 條件：section_type == 'documentary'
- 意見：`不爭執證據能力`
- 理由：書證物證的證據能力另依其他規則判斷，一般不爭執

### Rule 6: 被告以外之人的警詢筆錄
- 條件：record_type == '警詢筆錄' 且 sec_type != 'defendant_self'
- 意見：`被告以外之人於審判外之陳述，屬傳聞證據，無證據能力（刑事訴訟法第159條第1項）`
- 理由：警詢筆錄為審判外陳述，依傳聞法則無證據能力

### Rule 7: 秘密證人的偵訊筆錄
- 條件：witness_secret 且 sec_id 在秘密證人名單中 且為偵訊筆錄/偵查階段訊問筆錄
- 意見：`所述非親自見聞起訴書所載犯罪事實，屬傳聞證據，無證據能力（刑事訴訟法第159條之1第2項）`
- 理由：秘密證人所述內容為轉述他人所言，非親自見聞，屬再傳聞

### Rule 8: 被告以外之人的偵訊筆錄（三層判斷）

適用於偵訊筆錄或偵查階段的訊問筆錄，且為同案被告或證人的陳述。

#### 8a: 已合法調查（有審判筆錄）
- 條件：sec_id 在 PERSONS_WITH_TRIAL_TRANSCRIPT 中
- 意見：`被告以外之人於審判外向檢察官所為之陳述，屬傳聞證據，不同意作為證據（刑事訴訟法第159條之1第2項）`
- 理由：雖然該證人已於審判中作證（已合法調查），但偵訊筆錄仍屬傳聞，被告行使不同意之權利

#### 8b: 檢察官援引但未合法調查
- 條件：sec_id 在 PROSECUTOR_ADOPTED_PERSONS 中 但不在 PERSONS_WITH_TRIAL_TRANSCRIPT 中
- 意見：`經檢察官援引為證據方法，惟未經被告詰問之合法調查，不得作為判斷之依據（刑事訴訟法第155條第2項）`
- 理由：檢察官雖援引此筆錄，但證人尚未到庭接受被告詰問，未經合法調查

#### 8c: 檢察官未援引
- 條件：sec_id 不在 PROSECUTOR_ADOPTED_PERSONS 中
- 意見：`未經檢察官於補充理由書援引為證據方法，屬被告以外之人於審判外之陳述，為傳聞證據，不同意作為證據，無證據能力（刑事訴訟法第159條第1項、第159條之1第2項）`
- 理由：檢察官自己都未援引此筆錄，且屬傳聞證據

## 偵查階段判斷方法

卷宗字號是判斷筆錄屬於偵查階段或法院階段的關鍵：

| 字號關鍵字 | 階段 | 說明 |
|-----------|------|------|
| 偵字 | 偵查 | 檢察官偵查案件 |
| 相字 | 偵查 | 相驗案件 |
| 他字 | 偵查 | 他案偵查 |
| 國審強處 | 法院 | 國民法官審判之強制處分 |
| 訴字 | 法院 | 法院審理案件 |

## 同案被告的特殊處理

同案被告在本案中的地位為「被告以外之人」，其警詢筆錄及偵訊筆錄均適用傳聞法則。同案被告以證人身份於審判中作證時，該審判筆錄則不爭執。

## 法條引用格式規範

所有法條引用必須使用正式全稱：

| 正確格式 | 錯誤格式 |

| 刑事訴訟法第159條第1項 | 刑訴§159、刑訴§159I |
| 刑事訴訟法第159條之1第2項 | 刑訴§159-1II |
| 刑事訴訟法第155條第2項 | 刑訴§155II |

最高法院判決格式：`最高法院114年度台上字第3941號判決`
