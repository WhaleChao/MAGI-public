[ Melchior Setup Pack ]
=========================

請將此資料夾複製到您的 Windows 電腦 (Melchior) 上。

內容物：
1. DEPLOY_MELCHIOR.md  - 詳細安裝與設定說明書。
2. melchior_agent.py   - 工程師的大腦控制程式 (The Cerebellum)。
3. start_melchior.bat  - 一鍵啟動腳本。

快速安裝步驟：
1. 在 Windows 上建立資料夾 C:\AI\MAGI
2. 將 melchior_agent.py 放進去。
3. 參考 DEPLOY_MELCHIOR.md 安裝 Python 與必要的 Library。
4. 點兩下 start_melchior.bat 啟動。

啟動後，Casper 就可以自動控制工程師的算力模式了！
