# MELCHIOR 設定指南 (Setup Guide)

## 問題診斷
Casper 無法連接 Melchior 的 Ollama API (`http://100.116.54.16:11434`)。

## 修復步驟

### 1. 確認 Ollama 服務運行中
```powershell
# PowerShell
Get-Process ollama -ErrorAction SilentlyContinue
```
若未運行，啟動 Ollama：
```powershell
ollama serve
```

### 2. 修改 Ollama 綁定地址
Ollama 預設只監聽 `127.0.0.1`，需要改為 `0.0.0.0` 才能接受外部連線。

**方法 A：環境變數 (推薦)**
```powershell
# 設定環境變數
[Environment]::SetEnvironmentVariable("OLLAMA_HOST", "0.0.0.0:11434", "User")

# 重啟 Ollama
Stop-Process -Name ollama -Force -ErrorAction SilentlyContinue
ollama serve
```

**方法 B：修改 Ollama 服務設定**
如果 Ollama 以 Service 運行，編輯服務設定檔。

### 3. 開放 Windows 防火牆
```powershell
# 以管理員身份執行
New-NetFirewallRule -DisplayName "Ollama API" -Direction Inbound -Port 11434 -Protocol TCP -Action Allow
```

### 4. 驗證連線
在 **Melchior** 本機測試：
```powershell
curl http://localhost:11434/api/version
```

在 **Casper** 遠端測試：
```bash
curl http://100.116.54.16:11434/api/version
```
