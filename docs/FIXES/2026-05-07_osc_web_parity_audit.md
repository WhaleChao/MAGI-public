# 2026-05-07 OSC 網頁版功能對照

本次以 `/Users/ai/Desktop/Paperclip_rebuild/src/osc.py` 的單機版 Paperclip/OSC 為基準，對照 `/Users/ai/Desktop/MAGI_v2` 網頁版路由與前端。

## 已補齊

- 書狀「動手蓋」：單機版會顯示最後一頁並由使用者點選律師章位置。網頁版原本只有固定座標自動蓋章，本次已新增最後一頁預覽與點選座標，並串到單張蓋章與定稿合併。
- 定稿合併：網頁版已可產生正本、繕本、留底，合併同資料夾證據 PDF，並以 `原證1`、`附件二` 等檔名加註證據標籤。
- 報價單 PDF：已整合事務所 logo、名片、律師章等資源，並改成接近單機版報價單格式。

## 已在網頁版存在

- 案件列表、新增/編輯/刪除、案件資料夾、工作台、地址標籤。
- 行事曆、待辦事項、已完成狀態、PDF 掃描建立行程/待辦。
- 法扶管理、法扶流程、開辦/結案盤點、法扶文件查找。
- 書狀索引、書狀草擬、委任狀與收據產出、書狀蓋章、定稿合併。
- 實務見解、報價單、帳務、結案歸檔、備份/還原、PDF 工具、MAGI 三模組按鈕。

## 尚未完全等同單機版

- 狀頭狀尾修正：網頁版目前只有替換紀錄與模板資料，尚未做到單機版 `DocumentReplaceDialog` 的「讀取 Word 表格欄位、讓使用者逐欄確認、輸出新 DOCX」完整流程。
- 匯入模板 / 以既有書狀建立新書狀：網頁版有模板資料表，但尚未完全等同單機版從檔案直接匯入並產生新書狀的工作流。
- 案件卷夾標籤：網頁版已有法院/對造/法扶地址標籤，但單機版右鍵的「案件卷夾標籤」尚未做成同等按鈕。
- 最新法院通知：工作台可列出法院通知類文件，尚未做成單機版右鍵那種一鍵開啟最新法院通知。
- 假日管理：網頁版可掃描 PDF 並計算期限，但單機版的假日管理視窗、手動新增/刪除假日與 Google Calendar 假日同步尚未做成完整網頁設定頁。
- 常用詞熱鍵面板：網頁版已有書狀關鍵字資料維護，尚未做到單機版 F1-F10 快速複製、跟選定案件/文件連動的浮動控制台。
- 使用說明：單機版內建長篇操作說明；網頁版目前尚未整理成業務可讀的說明頁。
