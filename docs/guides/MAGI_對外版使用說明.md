# MAGI 對外版使用說明

最後更新：2026-03-12

這份是給一般使用者的對外版說明，只保留公開可用的常用功能。
管理員、維運、工程、資料庫、內部知識庫與系統除錯類指令，已另外隱藏。

## 1. 先知道這 3 件事

1. MAGI 最好用白話中文下指令，不需要學工程指令。
2. 如果你先上傳附件，接著只要回「翻譯」、「摘要」或「逐字稿」，MAGI 通常就知道你是在處理剛剛那份檔案。
3. 如果你不確定怎麼說，直接講目的就可以，例如「幫我翻譯這份 PDF」或「找關於詐欺的判決」。

## 2. 怎麼加入 MAGI

### LINE

1. 掃描管理者提供的 LINE 官方帳號 QR code。
2. 點管理者提供的 LINE 加好友連結。
3. 或者用管理者提供的 LINE 官方帳號名稱 / ID 直接搜尋。

正式對外資訊請管理者填入後再發送：

- LINE 加好友連結：<待補>
- LINE QR code：<待補>
- LINE 官方帳號名稱 / ID：<待補>

### Telegram

1. 在 Telegram 搜尋 MAGI 的 bot 使用者名稱。
2. 或直接打開 https://t.me/<bot_username>。
3. 如果是在群組裡使用，先把 MAGI Bot 加進群組，再開始互動。

正式對外資訊請管理者填入後再發送：

- Telegram Bot Username：<待補>
- Telegram 連結：https://t.me/<待補>

### Discord

1. Discord 多半不是加好友，而是加入已經有 MAGI 的伺服器。
2. 如果你只是使用者，向管理者索取 Discord 伺服器邀請連結即可。
3. 進入伺服器後，到指定頻道和 MAGI 互動。

正式對外資訊請管理者填入後再發送：

- Discord 伺服器邀請連結：<待補>

## 4. 基本操作

- 說明 / /help / /start / 有哪些功能：顯示功能總覽。
- 行程 / 今天有什麼 / 本週安排：查看近期安排。
- 查詢路由 翻譯 https://...：只看 MAGI 會怎麼判斷這句話，不真的執行。
- 幫我看這句會怎麼處理：翻譯 https://...：用白話方式說明路由結果。

## 5. 翻譯、摘要、搜尋、圖片、語音

### 文字與網頁

- 翻譯 [文字或網址]：直接翻成繁體中文。
- 完整翻譯 [文字或網址] / 不要摘要 / 全文翻譯：盡量全文翻譯，不做濃縮。
- 摘要 [文字或網址]：整理標準摘要。
- 短摘要 [內容] / 簡短摘要 [內容]：產出精簡版重點。
- 詳細摘要 [內容] / 長摘要 [內容]：產出完整重點整理。
- 摘要翻譯 [內容] / 翻譯摘要 [內容]：先理解內容，再做摘要式翻譯。
- 搜尋 [主題] / 網路研究 [主題]：上網找資料並整理。
- 抓取 https://... / 讀取網頁 https://...：直接讀指定網頁。
- 去AI味 [文字] / 潤稿 [文字] / 人性化 [文字]：把文字改得更自然。

### 圖片

- 畫一張 [描述] / /draw [描述]：生成圖片。
- 上傳圖片後說 描述這張圖：分析圖片內容。
- 上傳圖片後說 翻譯圖片文字：翻譯圖片裡的字。
- 上傳圖片後說 OCR：辨識圖片文字。
- 上傳圖片後說 上傳繳費憑證 / 這是繳費單：進入繳費憑證處理流程。

### 語音

- 直接上傳錄音檔：自動轉逐字稿。
- 逐字稿 /絕對路徑/audio.m4a：直接對指定音檔做轉錄。
- 上傳錄音後再說 翻譯：翻譯逐字稿。
- 上傳錄音後再說 摘要：摘要逐字稿。
- 台語 / 臺語：提示 MAGI 用台語與華語混合模式處理。

### 音樂

- 製作音樂 溫暖鋼琴、30秒：產生短音樂或配樂。

## 6. 附件怎麼用

### 上傳檔案後最常用的 4 句

1. 翻譯
2. 摘要
3. 不要摘要

- 翻譯與摘要：PDF、EPUB、TXT、MD、LOG、CSV、JSON、DOCX
- 語音轉錄：MP3、WAV、M4A、FLAC、OGG、WEBM

### 直接指定本機檔案的常見寫法

- 翻譯檔案 /Users/ai/Desktop/test.pdf
- 掃描檔命名 /Users/ai/Desktop/scan.pdf
- 逐字稿 /Users/ai/Desktop/audio.m4a

## 7. 法律文書草稿

這一類功能通常會進入對話式流程，MAGI 會一步一步問你需要的資訊，最後產生草稿。

- 幫我寫存證信函：建立存證信函草稿。
- 幫我做委任狀：建立委任狀或委託書草稿。
- 幫張三做民事委任狀：直接帶入姓名與案件類型。
- 幫我做委任契約書：建立委任契約書草稿。
- 幫我開收據：建立收據草稿。
- 取消 / 算了 / 不要：中途結束目前流程。

## 8. 閱卷與筆錄

### 閱卷

- 閱卷查核 基隆 114訴1：先查核案件，不直接送件。
- 檢查閱卷信箱：檢查是否有新的閱卷通知。
- 閱卷信件預覽 / 預覽閱卷：先看信件摘要。
- 可下載案件 / 可下載判定：判斷哪些案件可下載。
- 檢查閱卷信件與可下載案件：一次處理信件檢查與可下載判定。
- 下載閱卷：下載目前可下載案件。
- 下載閱卷 114年度訴字第123號：下載指定案件。
- 下載繳費單：下載待處理繳費單。
- 上傳繳費憑證：上傳單筆繳費憑證。
- 批次上傳繳費憑證：批次處理多張繳費截圖。

### 筆錄

- 同步筆錄 / 筆錄全同步：同步可下載筆錄。
- 重命名筆錄 / 筆錄更名：重新整理筆錄檔名。
- 下載全部筆錄 / 筆錄全下載：批次下載全部筆錄。
- 下載筆錄 114年度訴字第123號：下載指定案件筆錄。

## 9. PDF、判決與法規

### PDF 命名

- 掃描檔命名 /Users/ai/Desktop/x.pdf：替單一 PDF 做智慧命名。
- 單檔命名 /Users/ai/Desktop/x.pdf：和上面相同。
- 掃描檔放置區：批次處理待命名 PDF。
- 批次命名 / pdf scan：批次整理 PDF 檔名。

### 判決與法規

- 找判決 詐欺：依案由找判決。
- 查判決 113年度上訴字第12號：依案號找判決。
- 法規搜尋 刑法 第271條：查法規或條文內容。

## 12. 注意事項

- 如果是背景流程，例如閱卷、筆錄、下載、批次處理，通常不會當下完成，MAGI 會稍後回報結果。
- 如果你是用指定路徑的方式叫 MAGI 讀檔，路徑請盡量用絕對路徑。
- 如果你不確定 MAGI 會把一句話當成什麼功能，先用「查詢路由 [你的句子]」。
- 這份對外版沒有收錄管理員、維運與工程指令；若你需要進一步權限，請向管理者索取內部版說明。
