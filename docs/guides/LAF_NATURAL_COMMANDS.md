# CASPER 法扶回報自然語言命令集

更新時間：2026-02-21

## 原則
- 預設一律「只暫存，不送出」。
- 可用目標：`姓名`、`法扶案號`（例如 `1140728-K-002`）、`案件系統編號`（例如 `2026-0013`）。
- 若缺少目標，CASPER 會要求補充後才執行。

- `幫[當事人F]做疑義回報 原因資力不合標準`
- `[當事人E]訴訟中費用支付回報 說明裁判費2000`
- `[當事人L]二階段回報`
- `蔡旭欽結案回報 原因本案無閱卷必要`
- `王小明撤回回報 原因申請人撤回`
