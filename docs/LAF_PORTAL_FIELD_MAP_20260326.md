## 一、系統入口與導航

## 三、結案回報 (closing)

### 入口
- 搜尋頁面：`/lafcsp/toClosedReport`
- 表單頁面：`/lafcsp/toCR?tableName=PL41A`（POST 進入）

### 步驟流程
1. **填寫基本資料**（第一頁）
2. **結案意見回報**（第二頁，需先存檔第一頁才能進入）

### 第一頁：結案基本資料區

#### 頂部按鈕
- 存檔、下一頁、離開結案回報
- 引用、歷程

#### 結案基本資料區
| 欄位 | name | 型別 | 選項 |
|-----|------|------|------|
| 受扶助人於本案之身分 | `aidcdds` | select | 原告, 被告, 上訴人, 被上訴人, 告訴人, 聲請人, 抗告人, 相對人, 參加人, 其他 |
| 准予扶助種類 | - | readonly | |
| 扶助內容-程序 | - | readonly | |
| 對造姓名 | `opponent_name` | text | 有「範例說明」彈出按鈕 |
| 國民法官法程序案件 | `is_citizen_judge` | select | 是, 否 |
| 訴訟參與 | `is_litigation_participation` | select | 是, 否 |

#### 本案繫屬機關案號區（可新增多筆）
| 欄位 | name | 型別 | 選項 |
|-----|------|------|------|
| 繫屬院檢或其他機關 | `rel_court1` | select | 無, 不清楚, 法院, 檢察署, 其他 |
| 院檢名稱 | - | text+選擇按鈕 | |
| 裁判或其他決定之日期 | - | date | 年/月/日 |
| 本案案號 | - | text (年/字/號) | |

#### 本案相關結果情形區

##### 結案類型（彈出視窗，多層級聯動 select）

**第一層 casekd:**
- 扶助種類為訴訟代理或辯護
- 扶助種類為調解或和解
- 扶助種類為法律文件之撰擬
- 研究性法律諮詢

**第二層 level1（當 casekd=訴訟代理或辯護）:**
- 刑事案件
- 行政案件
- 民/家/勞案件
- 少年事件
- 憲法訴訟法程序（舊：聲請釋憲程序）

**第三層 level2（當 level1=刑事案件）:**
- 偵查程序
- 再議程序
- 請求非常上訴程序(偵查階段請求)
- 審判程序
- 交付審判
- 再審程序
- 非常上訴程序(審判階段聲請)
- 刑事補償程序(審判階段)
- 刑事補償程序(偵查階段)

**第四層 level3（當 level2=審判程序）:**
- 無罪判決
- 科刑判決
- 免訴判決
- 免刑判決
- 不受理判決(因和解或撤回告訴所致)
- 不受理判決(公訴)
- 撤回自訴或公訴
- 上訴駁回
- 原判決撤銷，另為無罪判決
- 原判決撤銷，另為科刑判決
- 原判決撤銷，另為免訴判決
- 原判決撤銷，另為免刑判決
- 原判決撤銷，另為不受理判決(和解或撤回告訴)
- 原判決撤銷，另為不受理判決(公訴)
- 原判決撤銷，發回原審法院
- 部分有罪，部分無罪判決
- 撤回上訴
- 其他

##### 其他欄位
| 欄位 | name | 型別 | 選項/說明 |
|-----|------|------|---------|
| 文件名稱 | - | readonly | 由結案類型帶入 |

## 八、共通上傳文件規格

| 項目 | 說明 |
|-----|------|
| **結案文件** | 限 *.docx / pdf / excel |
| **開辦文件** | 開辦通知書、委任狀等（無格式限制標示） |
| **費用/疑義/撤回附件** | + 上傳按鈕（無格式限制標示） |
| **撤回書** | 需至官網下載撤回書供當事人簽署，再上傳已簽署版本 |

## 九、共通按鈕行為

| 按鈕 | 行為 |
|-----|------|
